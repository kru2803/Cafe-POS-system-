# Cafe POS System

A simple Python-based point-of-sale (POS) system for cafes.

## Features
- Displays a menu to users
- Allows customers to place orders
- Calculates the total bill
- Applies a coupon-based discount
- Generates a final payable amount

## Tech Stack
- Python
- Git & GitHub

## Description
This project is a command-line based cafe order management system.
It uses Python dictionaries, loops, conditionals, and formatted output
to simulate a basic billing workflow.

## Purpose
This project was created as a learning exercise to strengthen Python
programming fundamentals and understand how a real-world system works.
